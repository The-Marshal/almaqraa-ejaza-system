# نظام إدارة طلبات الإجازة - منصة المقرأة 📖

مشروع تطويري لنظام المقرأة الإلكترونية، يهدف لتنظيم رحلة الطالب من تقديم الطلب حتى نيل الإجازة.

### 🛠 التقنيات المستخدمة:
* **Framework:** Laravel 10
* **Language:** PHP 8.1+
* **ORM:** Eloquent (with Eager Loading)
* **Standards:** PSR-12 (Formatted with Laravel Pint)

### 🚀 الميزات التي تم تطويرها في هذا الجزء:
1. **Query Optimization:** تم حل مشكلة (N+1 Query) عبر استخدام `with()` لجلب العلاقات، مما أدى لسرعة فائقة في معالجة البيانات.
2. **Advanced Filtering:** نظام بحث ديناميكي مرن يعتمد على `when()` و `whereHas()` للبحث في الجداول المرتبطة (مثل البحث بجنس الطالب أو دولته).
3. **Data Integrity:** استخدام الـ `Model Plucking` و `Collections` لتقليل استهلاك الذاكرة عند عرض الجداول الكبيرة.
